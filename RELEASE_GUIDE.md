# 下载指引 / Download Guide

## Android
- **armeabi-v7a**
  - 适用于较老设备（32 位 ARM）
- **arm64-v8a**（推荐）
  - 适用于大多数现代 Android 设备（64 位 ARM）
- **x86_64**
  - 适用于模拟器或少数 x86_64 设备

## Windows
- **windows-x64**（推荐）
  - 适用于大部分传统 PC / 笔记本（Intel / AMD）
- **windows-arm64**
  - 适用于 Windows on ARM 设备（如部分 Surface / Snapdragon PC）

## Linux
- **linux-x64**
  - 适用于大多数桌面 Linux 发行版（x86_64）

## macOS
- **macos-x64**
  - 适用于老款 Intel 芯片的 Mac
- **macos-arm64**（推荐）
  - 适用于 Apple Silicon 芯片的 Mac（M1 / M2 / M3 / M4 等）
